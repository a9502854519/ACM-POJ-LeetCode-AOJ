dp(i, j):前i頭牛工作前j秒的最小成本。
則我們有兩個選擇：
		1. 不雇用第i頭牛
			dp(i, j) = dp(i-1, j)
		2.雇用第i頭牛(T1 <= j <= T2)
			dp(i, j) = min( dp(i-1, k) + S | T1-1 <= k <= T2 )
兩者取最小值

優化：
dp(j):前j秒的工作成本
則對每頭牛更新:
		dp(T2) = min( dp(T2), min( dp(j)  | T1-1 <= j <= T2)  ) + S
		dp(0) = 0, dp(j) = INF | j > 0
查詢[T1, T2]區間最小值，可以用線段樹
